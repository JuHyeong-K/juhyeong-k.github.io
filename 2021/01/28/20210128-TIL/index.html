Parameter, Argument

parameter - 함수를 정의할 때 사용되는 입력 값
argument - 함수의 실행단위에서 쓰이는 입력 값

두 개의 차이를 잘 알아두자.

Arbitrary, Keyword Arguments

  • asterisk(*)하나로 표현
  • parameter에 정해지지 않은 여러 인자를 받을 수 있음
  • *args는 tuple 형태로 저장
  • asterisk(*)두개로 표현
  • parameter에 key-vaule 형식으로 여러 인자를 받을 수 있음
  • **kwargs는 dict 형태로 저장
