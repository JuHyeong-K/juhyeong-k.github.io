hexo로 블로그 만드는 방법

Posted on 2021-01-22 Edited on 2021-01-29 In etc

Requirments

  • Github ID
  • Node.js
  • Git

Github 호스팅용 저장소 생성

repository name은 domain으로 사용할 이름(juhyenog-k.github.io)으로 설정

hexo 설치

Node.js를 설치하면 사용할 수 있는 npm명령어를 사용해서 hexo를 설치한다.
$ sudo npm install -g hexo-cli

hexo설치가 끝난 뒤에 블로그로 사용될 폴더를 생성 후 그 디렉토리로 들어간다.
$ hexo init <folder>
$ cd <folder>

초기화에 필요한 패키지 설치
$ npm install

서버를 실행해서 확인해본다. 주소는 https://localhost:4000
$ hexo server

hexo 설정

hexo 블로그를 Github에 배포할 때 필요한 플러그인 설치
$ npm install hexo-deployer-git --save
package.json에 찍어준다

_config.yml파일을 사용해서 사이트 환경설정해줄 수 있다.
$ vi _config.yml
‘#Site에서 title, subtitle 등 정해주면 되고,’
‘#Deploy에서 type: git / repo: {Github 저장소 주소}’입력해주면 된다.
자세한 설명은_config.yml 설정 참조

post

원하는 제목()으로 post작성
$ hexo new post "제목"

저장된 경로 확인 (~/Documents/dev/blog/source/_posts/Test.md)

vim으로 해당 포스트 수정
~/Documents/dev/blog/source/_posts/Test.md

generate & deploy

$ hexo clean && hexo generate

  • hexo clean >> 캐시 파일 및 생성된 파일들(public)을 삭제
  • && >> 앞에 명령 실행완료 되면, 뒤에 명령 실행
  • hexo generate >> .md확장자로 만들어진 post를 html로 변환해주는 기능

서버를 다시 켜서 제대로 작성 됐는지 확인
$ hexo server

commit 작성 후, gitgub 저장소에 배포해주면 완료
$ hexo clean && hexo deploy
